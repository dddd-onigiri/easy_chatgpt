# browser_ttest

ブラウザ上でt検定→表→解釈まで出力できるウェブアプリです。

https://dddd-onigiri-browser-ttest-main-468zyy.streamlitapp.com/

Excel形式のデータをアップすることで簡単に検定が行えます。

GIGAスクールで一人一台端末が配備される中

タブレットでも分析ができたらいいなと思い、作りました。


開発環境

　Pycharm（Python3）

使用ライブラリ

    streamlit
    numpy
    pandas
    scipy
